# Documentación de Imagina CRM

Bienvenido a la documentación oficial del plugin **Imagina CRM** para WordPress. Aquí encontrarás guías paso a paso para configurar y sacarle provecho al producto.

> Imagina CRM convierte tu instalación de WordPress en una plataforma para gestionar listas, registros y automatizaciones tipo ClickUp/Airtable, todo nativo y sin servicios externos.

## Índice

2. [Listas](./listas.md) — qué son, cómo crearlas, configurarlas y eliminarlas.
3. [Campos](./campos.md) — los 14 tipos disponibles y cómo aprovecharlos.
4. [Slugs editables y URLs](./slugs.md) ⭐ — la característica única: cambia el nombre interno sin romper nada.
5. [Registros](./registros.md) — crear, editar inline, filtrar, ordenar y buscar.

7. [Acciones en bloque](./acciones-en-bloque.md) — selecciona múltiples registros y opera sobre todos a la vez.

9. [Preguntas frecuentes](./faq.md) — problemas comunes y soluciones.

## Convenciones

- Los pasos en negrita marcan **acciones** que el usuario debe ejecutar.
- Los `códigos` indican nombres internos que verás reflejados en URLs o en la base de datos.
- Las cajas ⚠️ marcan advertencias que conviene leer antes de ejecutar la acción.
- Las cajas 💡 marcan tips que aceleran tu trabajo.
